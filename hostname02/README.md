# Purpose
Program written in GO which spins up a webserver and prints its hostname.

# Getting help
    # make help

# Initialize module
    # make init

# Building
    # make build

# Checking syntax code
    # make lint

# Cleaning up
    # make clean

# Launching the server
    # ./go-hostname

# Making a request
    # curl localhost:8888

# Make each request returning a 500 code error
    # curl localhost:8888/trigger
