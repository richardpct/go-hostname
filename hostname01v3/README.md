# Purpose
Program written in GO that spins up a webserver and prints its hostname, after
5 hits it returns a code 500 error

# Getting help
    # make help

# Initialize module
    # make init

# Building
    # make build

# Checking syntax code
    # make lint

# Cleaning up
    # make clean

# Launching the server
    # ./go-hostname

# Making a request
    # curl localhost:8888
